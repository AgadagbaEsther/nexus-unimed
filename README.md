# Nexus-UNIMED: Interactive 3D Campus Map

**Author:** Agadagba Esther (Thesis Project)



### Technical Architecture

This application is built as a **Single Page Application (SPA)** using a modern web development toolchain:



* **Graphics Engine:** Three.js (WebGL) for 3D rendering and camera controls.

* **Build System:** Vite.js for optimized asset bundling and hot-module replacement.

* **Modeling:** GLTF/GLB high-fidelity models exported from Blender.

* **Deployment:** Automated CI/CD pipeline via GitHub Actions to GitHub Pages.



### Project Goals

1. To provide a high-performance 3D spatial interface for campus navigation.

2. To optimize 3D assets for mobile-web performance (Binary GLTF).

3. To implement a responsive UI for multi-device accessibility.
